 ## Arithmetic Operators

 |                          | Operator         | Example        |
 | :---:                    | :--------:        | :-------:     |
 | **Add**                  | **+**             | a + b         |
 | **Substract**            | **-**             | a - b         |
 | **Multiply**             | **&#42;**             | a * b         |
 | **Divide**               | **/**             | a / b         |
 | **Remainder**            | **%**             | a % b         |
 
 |                          | Operator          | Example           | Same As |
 | :---:                    | :--------:        | :-------:         | :------: |
 | **Increment**            |  **++**           |  a++              | a = a + 1
 | **Decrement**            |  **--**           |  a--              | a = a - 1|
